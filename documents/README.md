## Horizon User Documentation

Welcome to **Horizon**, a web framework written in Zig.

This directory contains user‑facing documentation for Horizon.
If you are *using* Horizon in your own Zig application, start here.

- **Overview**: [`overview.md`](./overview.md)
  High‑level overview, architecture, and technical requirements.
- **Getting Started**: [`getting-started.md`](./getting-started.md)
  Install Zig, set up the project, and run your first Horizon server.
- **Routing**: [`routing.md`](./routing.md)
  Define routes, path parameters, and mount route groups.
- **Request & Response**: [`request-response.md`](./request-response.md)
  Read queries/headers and build JSON/HTML/text responses.
- **Middleware**: [`middleware.md`](./middleware.md)
  Use logging, CORS, auth, static files, and write your own middleware.
- **Sessions**: [`sessions.md`](./sessions.md)
  Manage user sessions with in‑memory or Redis backends.
- **Templates**: [`templates.md`](./templates.md)
  Render HTML using ZTS (Zig Template Strings).
- **API Reference (High‑Level)**: [`api-reference.md`](./api-reference.md)
  Summary of types, methods, and built‑in components.
